Who am I?

My name is Bruno Domenico. I live and always lived in a spetacular city here in Brazil called Belo Horizonte (Beautiful Horizon in a rough translation). I actually want to describe the city but I should focus on describing me, as this section commands. Who am I? This is a portfolio page so this question should be easy to answer: I am a web and software developer looking for a myriad of opportunities. Opportunities for self development, for sharing, for caring, and for honest work. I should make a list here of my 'hard' skills, but that's why I have a github page, you should check that out if you haven't already. Maybe I should make a list of my soft skills, but you shouldn't trust any soft skills lists in the web, as we know, it could be fake. SO I HAVE FINALLY DECIDED to write here my special and unique skill set list, I hope you like it.
